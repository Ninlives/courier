#Courier

Yet another ActivityPub relay service

With this service provides relay endpoints for you to customize. In Mastodon's preferences go to:

  1. Administration
  2. Relay
  3. Click the button to setup a new relay
  4. Generate a relay address below
  5. Copy & paste into Mastodon's preferences

In Misskey's Control Panel go to:

  1. Relay
  2. Click the button to add a new relay
  3. Generate a relay address below
  4. Copy & paste into Mastodon's preferences
